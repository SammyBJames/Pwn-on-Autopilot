# Title: ZIP Drive

## Description
We had a big file that we wanted to compress so we gave it to Gary our new intern. Unfortunately, he though that every time he zipped it, it would get smaller... so he kept zipping the file over and over... and over. Can you recover our file from the archive?

**Note:** This challenge requires you to use a library we haven't discussed. Take a second to find a package that will give you the tools you need to solve the challenge.

## Artifacts
- `my_important_file.zip`: The outermost layer of the archive.
